# Voice Identity Developer Wiki

## Overview

This repository-hosted wiki is the maintainer reference for Voice Identity architecture, contracts, services, hardening validation, and release operations.

Voice Identity is a standalone Home Assistant custom integration with local-first, privacy-safe service surfaces.

## Current Release-Readiness Status

- Production-hardening artifacts VI-121 through VI-129 are documented and validated.
- HACS and hassfest workflows are defined in GitHub Actions.
- Release checklist and v0.1.0 draft notes are present.
- Final tagging/release is gated on passing GitHub Actions.
